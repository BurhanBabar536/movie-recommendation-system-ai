 # Movie Recommendation System

## Description
This project is a movie recommendation system built using machine learning. It suggests movies based on user input by finding similarities between movies using cosine similarity.

The goal of this project is to demonstrate how basic recommendation systems work using real data.

---

## Features
- Recommends similar movies based on selected movie  
- Simple user interface using Streamlit  
- Fast results using precomputed similarity  

---

## Technologies Used
- Python  
- Pandas  
- Scikit-learn  
- Streamlit

## How to Run

1. Clone the repository  
git clone https://github.com/your-username/movie-recommendation-system-ai.git  

2. Install dependencies  
pip install -r requirements.txt  

3. Run the application  
streamlit run app.py  

---

## Output
The user selects a movie from the list, and the system returns a list of similar movies.
